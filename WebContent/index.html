Final Project. 나의 정원
: 인터넷상에 생성된 나만의 정원에서 수확물 구독하기
“단순히 인터넷 쇼핑을 하는 것만이 아닌,
그곳에 나의 정원을 만들고 소유하는 건 어떨까?”
“가지고는 싶은데, 관리하기는 귀찮고...
그렇다면 공유 정원과 비밀 정원으로 공간을 나누자!”
⟪나의정원 둘러보기⟫
홈 계정 상품 공유정원 구독 비밀정원 게시판 관리자
  • 메인화면
  • 검색기능
  • 사이트 소개
  • 일반 로그인, 소셜 로그인
    • * 네이버, 카카오 로그인 API
    -소셜 로그인을 이용하면 번거로운 가입 과정을 거치지 않아도 사이트 이용 가능
    -회원가입 연동, 로그아웃, 회원탈퇴가 가능
  • 회원가입
    • -regex로 형식 검증 / ajax로 실시간 중복여부 검증
    -프로필 파일 확장자 검증
    -smtp로 발송된 인증번호 검증 후 나머지 양식이 slideDown됨
  • 마이페이지
    • -사용자의 모든 정보 제공
    -개인정보, 구매내역, 최근 구독내역 등을 열람할 수 있으며, 회원탈퇴 메뉴도 제공
    -상단 회색 박스의 정원이름, 프로필 사진은 ajax로 실시간 수정이 가능
    -소셜회원이 아닐 경우, 상세정보 수정은 비밀번호로 회원 검증
    * 마이페이지 개요화면
    -자바 캘린더를 이용한 달력 화면 구현(1년치 주문내역을 한눈에 / 주문 후 바로 반영)
    -Tooltip : 해당 칸에 대한 정확한 날짜를 보여줌
    -색 : 주문 수에 따라 칸의 색이 진해짐
    -녹색 박스 클릭 시, 간단한 주문정보와 상태가 slideDown으로 나타남
    * 주문내역 화면
    -주문내역에 관한 상세 정보를 확인할 수 있음
    -주문 상태와 조회기간에 따라 조회가능
    * 배송추적 화면(스윗트래커 API)
    -스윗트래커 API를 이용한 배송 추적 기능 구현
  • 상세페이지
    • -fixed된 주문서로 언제든 상품수량을 정하고 장바구니에 담을 수 있는 기능 구현
    -상세페이지 하단에 질문과 후기, 댓글을 쓸 수 있는 기능 구현
    -‘도움돼요’를 누르면 추천수가 올라가고, 추천수가 높은 후기는 메인화면에 노출
  • 장바구니
  • 공유정원
  • 구독하기
    • -정기구독 선택 화면(주문화면)
    -아임포트 API를 이용한 카드결제, 무통장 입금 기능 구현
  • 비밀정원
  • 자유게시판
    • -레시피나 질문 글을 쓸 수 있는 자유게시판 기능 구현
    -각 글마다 댓글 쓰기, 수정 기능 구현
    -페이징 기능으로 깔끔한 레이아웃 구현, 검색 기능으로 원하는 글 검색 가능
  • 관리자페이지
    • -회원관리, 상품관리 등 각종 관리가 가능한 페이지
    -chart.js : 총 판매금액, 취소내역, 수익 등 사이트 운영에 필요한 통계 제공
    -transaction, trigger : 트랜잭션과 트리거를 이용한 주문, 배송내역 관리
     나의정원 시연영상

    Semi Project. 우야꼬
    : 디지털 소외계층을 위한 무인주문기 체험 사이트
    “디지털 소외계층이 날로 늘어가는 지금,
    매장 곳곳에는 무인주문기가 보급되고 있다”
    “무인 주문기 사용을 어려워하는 사람들을 위해
    기기 사용을 체험하고 학습할 공간을 마련해주면 어떨까?”
    ⟪WUYAKO 둘러보기⟫
    홈 소개 체험하기 유튜브 검색 마이페이지 게시판 관리자
  • 무인주문기 체험을 하며 원리를 학습할 수 있는 페이지
    • -지시사항을 이행하게끔 도와주는 포인터를 자바 스크립트로 구현
    -게임 시작 시, 난수로 이루어진 코드 부여
    -게임 종료 후, 코드는 삭제되나 회원일 경우 마이페이지에 결과 저장
    -TTS API를 사용해 텍스트를 음성으로 치환시켜주는 기능 구현
    -무인주문기 체험을 위한 UI 구현
    -중복을 제거한 로직으로 매 게임마다 메뉴 배치가 바뀜
    -Selenium을 이용한 실시간 메뉴 업데이트 가능
    -팝업으로 추가 정보 전달
    -최종 결과를 확인할 수 있는 페이지 제공
    -음성, 문자인식 검색기능 제공
    -가까운 매장 찾기, 지도 찾기 기능 제공
  • 유투브 검색을 하며 더 많은 정보를 접할 수 있는 페이지
    • -구글 디벨로퍼스 API
    -키워드로 유튜브 영상 실시간검색 가능, 더 많은 정보에 접근 가능
    -페이징 기능으로 편리한 탐색 기능
  • 회원들이 자유롭게 글을 올릴 수 있는 자유게시판
    • -페이징 기법을 통한 게시판 글 목록 및 페이지
    -네비게이터와 검색기능 구현
    -댓글 등록 기능 및 삭제, 수정 기능 구현
    -관리자의 답변 작성 유/무 확인 기능
    -썸머노트를 활용하여 이미지를 첨부한 게시글 등록이 가능
  • 사이트 관리, 회원관리, 통계 확인을 할 수 있는 관리자 페이지
    • -스케줄러를 활용, 매일 자정에 방문자 수 업데이트
    -유저관리를 통한 불량 유저차단 및 복원 가능
    -차트 JS를 이용한 최근 한달 방문자 수 및 성별 누적 통계
    -연령대별 회원 수 통계
    -관리자가 게시판 관리 및 답변 작성 가능

    Study. 지옥으로 가는 git

    “버전관리로 프로그래밍 공부도 하고,
    gitHub 잔디밭까지 관리하기!”
    click!

    These are common Git commands used in various situations:
    start a working area (see also: git help tutorial)
    clone Clone a repository into a new directory
    init Create an empty Git repository or reinitialize an existing one
    work on the current change (see also: git help everyday)
    add Add file contents to the index
    mv Move or rename a file, a directory, or a symlink
    reset Reset current HEAD to the specified state
    rm Remove files from the working tree and from the index
    examine the history and state (see also: git help revisions)
    bisect Use binary search to find the commit that introduced a bug
    grep Print lines matching a pattern
    log Show commit logs
    show Show various types of objects
    status Show the working tree status
    grow, mark and tweak your common history
    branch List, create, or delete branches
    checkout Switch branches or restore working tree files
    commit Record changes to the repository
    diff Show changes between commits, commit and working tree, etc
    merge Join two or more development histories together
    rebase Reapply commits on top of another base tip
    tag Create, list, delete or verify a tag object signed with GPG
    collaborate (see also: git help workflows)
    fetch Download objects and refs from another repository
    pull Fetch from and integrate with another repository or a local branch
    push Update remote refs along with associated objects
